Необходимо выбрать из 4 дополнительных карточек ту, которая соответствует целевой

Только 1 из признаков может быть верным для целевой карточки
Для выбора карточки необходимо щелкнуть на нее ЛЕВОЙ КНОПКОЙ МЫШИ
Вам сообщат, выбрали ли Вы верно или неверно
